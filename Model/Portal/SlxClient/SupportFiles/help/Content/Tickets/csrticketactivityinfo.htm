Ticket Activity Information Boxes

The following table describes the default information boxes.

Box

Description/Action
Account Displays the account associated with the ticket activity. Click the account name to open the Account Detail view.
Contact Displays the contact associated with the ticket activity. Click the contact name to open the Contact Detail view.
Ticket Displays the ticket associated with the ticket activity. Click the ticket name to open the Ticket Detail view.

Type

The kind of work performed on the ticket. The type "Timed" appears by default if this a timed activity (Punch In/Out). To edit, click theellipsis button and select an item from the list. Click OK.

Public Access

Viewing access for the ticket activity. This determines whether or not the customer can view the ticket activity using Customer Portal. To edit, click theellipsis button and select an item from the list. Click OK.

User

The name of the logged-in user. Click the Find button and edit using the lookup.

Start Date

The date and time the user began the activity. This is important for charging time against the customer's contract. To edit, click the Calendar/Time button, select the date, and then click OK.

End Date

The date and time the user completed the activity. This is important for charging time against the customer's contract. To edit, click the Calendar/Time button, select the date, and then click OK.

Follow Up

Indicates whether the ticket is marked for follow-up based on the ticket activity. Selecting the check box will add the Ticket to the Follow-Up group in the Ticket List view.

Time Units

The number of time units being charged against the customer's account. The system updates this box when you enter the end date or punch out of the ticket.

Elapsed Hours

The time that passed between the start and end dates or between the time punched in and the time punched out. The system updates this box.

Charge Type

The price to be charged to the customer for the activity. Click the Find button and edit using the lookup.

Rate

Displays the activity rate amount  for the selected Charge Type. The system supplies this number based on the Charge Type selected.

Total Labor

Displays the total labor charge for the activity. The system calculates this amount from the Rate multiplied by the Elapsed Hours.
Total Parts Displays the total parts charge for the activity. The system calculates this amount from the cost of all associated ticket activity parts.
Total Fee Displays the total charge for the activity. The system calculates this amount by adding the Total Labor and Total Parts.

Comments

Contains details of the activity. In the text box, type or paste text copied from another source.
