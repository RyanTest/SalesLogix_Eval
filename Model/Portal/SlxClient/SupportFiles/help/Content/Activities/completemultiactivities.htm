Completing Multiple Activities

You can complete multiple activities from the Activity Reminders view. You can either complete multiple activities at the same time, or complete each activity separately. If you choose to complete each activity separately, the Complete Activity view will open allowing you to edit activity information before completing it. Once you complete an activity, the Complete Activity view will open for the next selected activity until each of the selected activities have been completed.

To complete multiple activities at once

  1. Open the Activity Reminders view.
    • Do one of the following:
    • From the Schedule menu, click Activity Reminder.
    • In the Menu Bar, click Reminders.
  2. Select the check box for each activity you want to complete, then click Complete Selected Activities.
  3. Click the Result ellipsis button and select an item from the list.
  4. In the Note field, add any information that you want to be added to all of the activities you are completing.
  5. Click As Scheduled to complete the activities using the date and time you entered when you scheduled the activities. Or, click Now to complete the activities using the current date and time.

To complete multiple activities individually

  1. Open the Activity Reminders view.
    • Do one of the following:
    • From the Schedule menu, click Activity Reminder.
    • In the Menu Bar, click Reminders.
  2. Select the Select checkbox for each activity you want to complete, then click Complete.
  3. Click Individually.
  4. Follow the remaining steps for completing each activity until each activity has been completed.
