Completing an Activity - Mail Merge

When you perform a mail merge activity, you will be prompted to complete the activity. When you complete an activity, details about it are recorded in the History tab.

To complete a mail merge activity

  1. If necessary, modify the Completed date or time. To modify the date or time, click the Calendar/Time button, and then select the date and time the activity was completed.
  2. If the activity did not occur at a particular time, select the Timeless check box instead.
  3. If necessary, modify the Scheduled date or time. To modify the date or time, click the Calendar/Time button, and then select the date and time the activity was scheduled.
  4. In the Duration boxes, enter a number and select Minute(s), Hour(s), or Day(s) from the drop-down list to determine the length of time the activity requires.

The duration of the completed activity is not recorded to history; only the completed time is recorded to history.

  1. If necessary, specify a follow-up activity in the Follow-Up box. Select the Carry Over Notes or Carry Over Attachments check box to copy notes or attachments from this activity to the follow-up activity.
  2. If necessary, modify the Regarding, Category, and Result boxes. By default, these boxes contain information from the mail merge. To edit the information boxes, ellipsis button and select an item from the list and then click OK.
  3. In the Notes box, type any additional information about this activity.
  4. If another user completed the activity, click the User Find button, select the user from the list, and then OK.
  5. Click the Attachments tab to add a document or URL to the activity.
  6. Click Complete.

If you scheduled a follow-up activity, the Schedule an Activity dialog box appears for you to complete.
