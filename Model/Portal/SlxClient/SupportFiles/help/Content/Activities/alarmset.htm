Setting Reminders

Reminders alert you to an activity that is due to be completed. Reminders are scanned based on the options selected in your user preferences. By default, the Web Client scans your activities for reminders every 5 minutes. You can set a reminder for an existing activity or when you schedule an activity.

To set the alarm

  1. Do one of the following:
  • Schedule a new activity.
  • Find the activity for which you want to set a reminder, and open the Activity Detail view.
  1. Select the Reminder check box, type the amount of time and then select the interval of time at which you want to be reminded of this activity. You can select minutes, hours, or days.

To set alarm and reminder defaults

  1. On the toolbar, click Options, and then click the Activity/Alarm tab.
  2. In the Alarm Options area, select the appropriate choices. You must enable alarms to use them.
  3. Click Save.
