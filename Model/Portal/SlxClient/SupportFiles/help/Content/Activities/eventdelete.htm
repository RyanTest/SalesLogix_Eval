Deleting an Event

You can delete any event you create from the Activity List or Calendar view

To delete an event

  1. Open the Activities List view.
  2. Click Events.
  3. From the Users list, click the user(s) for whom you want to view events.

To select multiple users, hold down the CTRL key while you click the users in the list.
  1. Click the Time Frame drop-down arrow, and select a time frame.
  2. Click the Event Start Date link.
  3. In the Event Detail view, click the Delete button.
  4. Click OK to permanently delete the record, or Cancel to cancel deletion of the record.
