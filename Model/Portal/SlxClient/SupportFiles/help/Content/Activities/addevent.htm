Scheduling/Editing an Event

Events can be scheduled, edited, deleted, and viewed in the Web Client.

If you are using Intellisync for SalesLogix to synchronize activities and events with Microsoft Outlook or Lotus Notes, duplicates will be generated under the following combination of circumstances:

  • you create the event in Sage SalesLogix, and
  • synchronize the event to Outlook or Lotus Notes, and then
  • in Outlook or Lotus Notes, change the event to a recurring event.

When the event synchronizes back to SalesLogix, it will appear as a recurring timeless activity, but the original event will also be on the calendar. To prevent confusion, you can manually delete the original event. See the topic called Deleting an Event for more information.

 

To schedule an event

  1. On the Schedule menu, click Event.
  2. In the Start Date box, modify the date. Click the Calendar/Time button, select the day on which you want to schedule the event, and then click OK.

If you do not use the default start date, once you have entered the new start date, the end date automatically updates to match the start date.

  1. If necessary, modify the End Date. Click the Calendar/Time button, select the day on which you want to schedule the event, and then click OK.
  2. In the Day Type box, click the ellipsis button and select the type of event.
  3. In the Description box, type information about the event.
  4. Click the User Find button, select the user for whom the event is being scheduled, and then click OK.
  5. Click Save.

 

To view or edit an event

  1. Open the Activities List view, and then click Events.
  2. Select the user whose events you want to view, and in the Time Frame box, select the appropriate range (such as Today or Last Month).
  3. Click the event Start Date to open an event.
  4. If necessary, change the event information.
  5. Click Save.
