Product Filter Options

Use Filter options to limit the products that display in the list of available products. You can apply a filter by selecting the check box. The filters will retain your last selection as the default values. You can select from the following filters:

  • Name and SKU filters - These fields allow you to enter a condition.
  • Family filter - Click the ellipsis button and select an item from the list.
     
  • Status filter - Click the ellipsis button and select an item from the list.
     
  • Display product packages only - Selecting this option will display product suites and their contents. When selected, this filter will disable the SKU, Family, and Status fields, and change the Name label to Package Name.

To select filter options

  1. Select the check boxes of the filters you want to apply.
  2. Enter or select the appropriate conditional information.
  3. Click Show Results.

To clear filter options

  • Click Clear Filters.
