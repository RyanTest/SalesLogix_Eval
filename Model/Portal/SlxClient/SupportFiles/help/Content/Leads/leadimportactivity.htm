Lead Import Activity Information Boxes

Use this view to add an activity to each imported lead. The full text of the activity, date and time appears in the Activities tabs for the Detail views.

To schedule

  1. When importing leads, create a group action and select Schedule Phone Call, Schedule Meeting, or Schedule To-Do.
  2. For the Start Time, click the Calendar/Time button to select a date, and then click OK.
  3. If the activity does not need to occur at a specific time, select the Timeless check box.
  4. For the Duration, enter a number and select Minute(s), Hour(s), or Day(s) from the drop-down list to determine the length of time the activity requires.
  5. Select the Auto Rollover to Next Day check box to automatically roll over the activity to the next day if it is not completed on the scheduled date. This option is only available if Timeless is selected.
  6. If you want to be reminded of the activity before it occurs, select the Reminder check box, type a number in the box, and select Minute(s), Hour(s), or Day(s) from the drop-down list to determine the length of time the reminder should occur before the activity start time.
  7. In the Regarding box, click the ellipsis button and select an item from the list.
  8. In the Priority box, ellipsis button and select an item from the list.
  9. In the Category box, ellipsis button and select an item from the list.
  10. In the Notes area, type any additional information for the activity.
  11. Click Save.

The activity leader will be determined for each lead once it is imported.
