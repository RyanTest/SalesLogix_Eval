Importing Leads from a List

Use the Import Leads view to import lists of leads into Sage SalesLogix.

The text files must be either ANSI or ASCII delimited text files.

To import leads

  1. From the Tools menu, click Import Lead.
  1. Complete the Select a File information and click Next.
  2. Complete the Define a Delimiter information and click Next.
  3. Complete the Map Fields information and click Next.
  4. Complete the Manage Duplicates information and click Next.
  1. Complete the Select Import Actions information and click Next.
  2. Review Import selections and click Submit.
  3. Monitor the Import Process Request.
