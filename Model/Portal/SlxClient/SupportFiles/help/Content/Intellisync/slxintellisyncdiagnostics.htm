Reviewing Intellisync for SalesLogix Diagnostics

You can access Intellisync for SalesLogix diagnostics from the Intellisync portal and the Sage SalesLogix Personal Web Server. Diagnostics can be used to troubleshoot Intellisync for SalesLogix installations.

To access diagnostics from the Intellisync portal

  1. Open the default page in your Intellisync for SalesLogix portal. Enter the page in your browser using the format: http://server:port/portal name/default.aspx
  2. Click Diagnostics.
  3. Do one of the following:
  • To view assemblies and their version number, click the forward and back buttons at the bottom of the list.
  • To test user credentials, enter the User Name and Password for the user you want to test and then click Try to Connect.

A request is submitted to the server to test if the user can log on to the Intellisync portal. A message indicates if the test was successful.

To access diagnostics from the Personal Web Server

  • To test user credentials:
  1. Right-click the Sage SalesLogix Personal Web Server icon in the system tray, and click Properties.
  2. In the Sage SalesLogix Personal Web Server dialog box, click Manage.
  3. Enter the User Name and Password for the user you want to test.
  4. Click Test Connection.

A request is submitted to the server to test if the user can log on to the Intellisync portal. A message indicates if the test was successful.

  • To view assemblies and their version number:
  1. Right-click the Sage SalesLogix Personal Web Server icon in the system tray, and click Properties.
  2. In the Sage SalesLogix Personal Web Server dialog box, select the Intellisync portal in the Web Site box.
  3. Click Details.

View the installed assemblies for the Intellisync portal and their version number.

  • To compare portal versions:
  1. Right-click the Sage SalesLogix Personal Web Server icon in the system tray, and click Side by Side Versions.
  2. View the version of the sage.saleslogix.dll file for each installed portal.
