Deleting Activities, Contacts, or Events

Intellisync for SalesLogix does not allow Outlook or Lotus Notes to delete activities, contacts, or events from the Sage SalesLogix database.  You must delete these records manually after synchronizing.

Marking Deleted Items

When an item is removed from Outlook or Lotus Notes the tag ISYNC DEL is added to the Sage SalesLogix activity, contact, or event record as follows:

  • For activities, the ISYNC DEL tag is inserted in the Priority field.
  • For contacts, the ISYNC DEL tag is inserted in the Description field.
  • For events, the ISYNC DEL tag is inserted in the Type field.
Viewing Deleted Items
  • To see all activities that were removed from Outlook or Lotus Notes sort the Activities List by Priority code and search for ISYNC DEL.
  • To view all contacts that were removed from Outlook or Lotus Notes sort the Contacts List by the Description field and search for ISYNC DEL.
  • To view all events that were removed from Outlook or Lotus Notes sort the Events List by the Type field and search for ISYNC DEL.

Manually Deleting a Tagged Item

You can delete the activities, contacts, or events from Sage SalesLogix if desired.

  1. Locate the item as detailed in Viewing Deleted Items earlier in this topic.
  2. Select the item, right-click, and then click Delete.
