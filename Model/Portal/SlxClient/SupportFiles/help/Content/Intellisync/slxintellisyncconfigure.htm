Configuring Intellisync for SalesLogix

If your company is using Intellisync for SalesLogix, you must configure synchronization preferences. This configuration must be completed on each user’s computer. The following steps outline the recommended synchronization settings between Microsoft Outlook or Lotus Notes and Intellisync for SalesLogix.

To configure Intellisync, you will need the URL for the Intellisync for SalesLogix portal.  Contact your system administrator for more information.

The Intellisync setup interface Help buttons call topics created by Intellisync's creator, Nokia. Some of the terminology in these help topics may differ from the terms used in a Sage SalesLogix implementation.  

To configure

  1. On the Start menu, point to Programs, point to Sage SalesLogix, point to Intellisync for SalesLogix, and then click Intellisync for SalesLogix.

  1. Click Setup.

  2. In the User ID box, type your Sage SalesLogix user name.

  3. In the Password box, type your password.

  4. In the Connection URL box, type the path to your Intellisync for SalesLogix portal.

Contact your system administrator for the URL to the portal location.

  1. Click Application Setup.

  2. On the Setup screen, select the type of data you want to synchronize (for example, Appointments), and then click Setup Wizard.

  3. On the Desktop Application screen, select Microsoft Outlook or Lotus Notes, and click Next.

  4. On the What to Sync screen, select All Intellisync for SalesLogix applications that are supported by Microsoft Outlook or Lotus Notes, and click Next.

  5. On the How to Sync screen, select Two way sync, and click Next.

  6. On the When To Sync screen, select when you want synchronization to occur, and click Next.

  1. On the Connection Information screen.
  • If you are configuring Outlook, select your user profile, and click Next.
  • If you are configuring Lotus Notes, type your Lotus Notes password, and click Next.
  1. Complete additional screens as appropriate for the type of information you are synchronizing for either Outlook or Lotus Notes.
  1. On the Options for ToDo’s screen for either Outlook or Lotus Notes, select Transfer all items.

    This options synchronizes all active, non-completed, and completed ToDo’s.

  1. If you are configuring Intellisync for Lotus Notes, on the Options for Contacts screen, browse to the Lotus Notes database and click Next.
  2. On the Options for Appointments screen for either Outlook or Lotus Notes, select Transfer only future items.
  1. On the Intellisync for SalesLogix Connection Information screen, verify your Sage SalesLogix user ID, password, and the connection to your Intellisync for SalesLogix portal, and then click Next.
  2. Click Finish.
  3. Click Close.
  4. Click OK.
  5. If you have not already done so, you must select the SyncSalesLogix group.
