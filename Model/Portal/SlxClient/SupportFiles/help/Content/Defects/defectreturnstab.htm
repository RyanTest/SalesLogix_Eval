Using the Defect Returns Tab

Use this tab to add or remove an association between returns and defects. Management and other users may use this information to determine the severity of the defect and actions needed to correct the defect.

To open

  1. Open the Defect Detail view.
  2. Click the Returns tab. If the tab is not visible, click the More Tabs tab.

To associate

  1. Click Associate Return.
  2. Locate and select the return using the lookup.
  3. Click OK.

To disassociate

  1. Select the item you want to remove, and then click Disassociate.
  2. Click Yes.
