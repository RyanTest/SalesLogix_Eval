Defect Details Tab

The Details Tab appears in the Defect Detail view. This tab contains specific information about the defect.

To edit the details tab

  1. From the Defect Detail view, click the Details tab.If the tab is not visible, click the More Tabs tab.
  2. In the Subject box, type a brief description on the problem.
  3. Use the Description box to type a detailed description of the problem.
  4. Use the Resolution box to type the steps taken to resolve the defect.
  5. Click Save.
