Adding a Defect

Use the Insert New Defect view to create a defect record. Use the copy button to quickly create a new record with similar information.

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

To add a defect

  1. Open the Insert New Defect dialog box.
    • On the New menu, click Defect.
    • On the Ticket Detail view, click the Defects tab, and then click the Create Defect button.

     

  2. Complete the required defect information boxes.
  3. Click Save.
  4. If necessary, add defect tasks and any associated products, returns, or tickets using the Defect Detail view tabs.
