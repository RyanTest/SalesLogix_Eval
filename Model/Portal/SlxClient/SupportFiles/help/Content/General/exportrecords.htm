Exporting Records to a File

You can export records to a file. The group column headings and formatting will also be exported.
  • Export to File requires the use of Active X Mail components. Active X is not supported by Firefox; therefore, this feature is only available with Internet Explorer.
  • If the Export to File option is unavailable, you will need to download the ActiveMail components. The ActiveMail components are downloaded when you log on to the Web Client. In order to download these components, you can set the ActiveMail option in Options, or on the Sage SalesLogix Web Client login page.

 

To export a group to a file

  1. Open the group that contains the records you want to export.
  2. Select one or more records by pressing the CTRL or Shift key as you click each item.
  3. From the Task Pane, select Export to File.

If you do not select any records, you will be asked if all records in the group should be used. If you click OK, all group records will be selected.
  1. Select which file format you would like to use.
  • Standard CSV - individual records are separated by a comma
  • Tab Delimited- individual records are separated by a tab

If you want Sage SalesLogix to remember your selection and you do not want to be prompted again, select Remember my selection and do not ask again.

  1. Click OK.

You may receive an error if you try to export a group that includes a data field with a large amount of data.
