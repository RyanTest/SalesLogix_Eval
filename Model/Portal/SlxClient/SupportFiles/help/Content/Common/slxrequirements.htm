Sage SalesLogix Web Client Requirements

To use the Sage SalesLogix Web Client, you must use at least:

  • Microsoft Explorer 6.0 or higher or Firefox 3.0.

Your browser must have the following settings enabled:

  • Accept cookies.
  • Check for new versions of the page every visit to the page.

To run Web reports in a disconnected state

  • The disconnected Web Client must run on Window 2003 Server with IIS.
  • The SalesLogix Web Reporting Server and Crystal Reports Server must also be installed and configured.

If these requirements are not met, disconnected Web Client users can only run Web Reporting if they have a connection to the Host Web Reporting Server. These users will see reports that represent data from the host database and not their remote database.
