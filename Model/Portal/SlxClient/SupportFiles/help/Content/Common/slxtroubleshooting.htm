Troubleshooting

The following are commonly experienced issues:

My queries all return the same data and I keep getting the same pages.

Your browser is not processing certain tags correctly and is pulling pages from your cache (a folder that stores Web pages) instead of getting pages from the server as you use the Web Client. You need to set your browser viewing options to make sure it pulls data from the server and not your cache.

To fix this problem in Microsoft Internet Explorer
  1. In Microsoft Internet Explorer version 6.0 and later, on the Tools menu, click Internet Options, and then click the General tab.
  2. In the Temporary Internet Files section, click Settings.
  3. In the Check for newer versions of stored pages section, click Every visit to the page.
  4. Click OK twice to save your changes.
To fix this problem in Firefox
  1. In Firefox version 3.0 and later, on the Tools menu, click Options.
  2. Click Privacy.
  3. In the Private Data section, click Settings.
  4. Select Cache and any other types of private data to be erased when you clear private date.
  5. Click OK.
  6. (Optional) To clear your cache every time your browser is closed, select the Always clear my private data when I close Firefox option.
  7. Click the Clear Now button.
  8. Click OK.

the Sage SalesLogix Web Help Behaves Differently in Firefox

If you are using Firefox 3.0 as your browser, the Sage SalesLogix Web Help opens in a new tab in the same browser window and resizes the window. However, you can set options that will prevent the window from resizing or open any new pages in a separate window.

Changing these options, will impact the behavior of tabs and windows for all Web sites, not just Sage SalesLogix pages.
To prevent the window from resizing
  1. In Firefox version 3.0 and later, on the Tools menu, click Options.
  2. Click Content.
  3. Next to Enable Javascript, click Advanced.
  4. Clear the option to Move or resize existing windows and click OK.
  5. Click OK.

 

To open all new pages in a separate window
  1. In Firefox version 3.0 and later, on the Tools menu, click Options.
  2. Click Tabs.
  3. Under New pages should be opened in section, select a new window.
  4. Click OK.
