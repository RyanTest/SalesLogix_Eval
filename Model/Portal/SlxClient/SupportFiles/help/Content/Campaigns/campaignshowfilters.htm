Campaign Target and Response Filters

Use the Show Filters options to limit the target contacts and leads that display in the list.

To filter

  1. Click Show Filters to expand the list of filters.
  2. Select the check box(es) of the filter(s) you want to apply.
  • Show Contacts - Displays contact responses that meet the filter requirements.
  • Show Leads - Displays lead responses that meet the filter requirements.
  • Lead Source - Filters target responses by the response lead source.
  • Response Method - Filters target responses by how the target responded to the campaign.
  • Stage - Filters target responses by the response stage.
  • Last Name - Filters target responses by target last name.

You may select more than one filter option. For example, you can check the Show Contacts and Response Method options to view contact responses that have the same response method.

  1. If necessary, click the Refresh button to update the grid.
