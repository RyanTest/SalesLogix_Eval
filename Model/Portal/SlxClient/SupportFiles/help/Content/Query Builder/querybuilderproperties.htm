Query Builder Properties

Use the Query Builder Properties tab to enter the name and purpose of a query.

To set Query Builder properties

  1. Open Query Builder.

  2. Click the Properties tab.

  3. In the Name box, type a name for your query.

    4. If you are creating a group, the name you give the query appears in the Group Manager.

  4. In the Display Name box, type the name as you would prefer it to be displayed.

    If you are creating a group, the display name you give the query appears on the Group tab within the list view.  If no Display Name is entered, the tab will display the text entered in the Name field.

  5. In the Description box, type a description to identify the purpose of the query.
