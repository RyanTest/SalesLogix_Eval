Left Joins

A left join is the default type for global joins. Use a left join to include all the records in the Parent table that meet the selection criteria and only those records in the Child table where the values in the fields you are joining are equal.

Example: You want a list of all opportunities whether or not they have associated activities. If the opportunity is associated with an activity, select it as well. The records shown in the Join Results column are returned by this query. The specific opportunity associated with an activity is indicated in parentheses to the right of the activity.
