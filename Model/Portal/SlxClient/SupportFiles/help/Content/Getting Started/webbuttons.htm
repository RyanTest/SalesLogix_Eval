Understanding the Sage SalesLogix Web Client Workspace

The workspace is the area where various views appear in the main Sage SalesLogix Web Client window.

Each view in your Sage SalesLogix Web Client contains many of the following elements.

Your workspace may be different depending on your access rights or if your installation uses a customized Sage SalesLogix Web Client.

 

Element

Description

Menu Bar

The menu bar contains the Sage SalesLogix Web Client commands. These include Recently Viewed, Help, New, Schedule, Write, Opportunities, and Groups menus, Reminders button and the SpeedSearch button.

Click a menu to expand it.

Navigation Bar

Located on the left side of the Sage SalesLogix Web Client workspace, the Navigation Bar containsbuttons that open the main Sage SalesLogix Web Client views. Your administrator can group the buttons into subsets that focus on specific elements of your business, such as Sales or Support.

Title Bar

The title identifies which record type is currently being displayed in the active view. When you are in the Account, Contact, or Opportunity Detail views, the Title Bar displays the name of the current account, contact or opportunity.

Tabs

Tabs organize information on many of the detail views and dialog boxes.

Status Bar

The status bar displays the user and current date. It also contains the link to Options, where you can select from available options for e-mail, calendar and alarms, set a default group and record type, change your password, and enter default settings for new opportunities.

Task Pane

The Task Pane displays to the right of the main window pane. The Task Pane consists of common tasks and features designed to help you complete your work. The available options depend on the record type and type of view where you are working.
