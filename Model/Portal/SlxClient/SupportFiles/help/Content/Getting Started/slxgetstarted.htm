Getting Started with the Sage SalesLogix Web Client

The Sage SalesLogix Web Client helps you manage all aspects of the sales cycle over the Web. This topic includes links to commonly performed tasks and overview topics to help you begin to use the Sage SalesLogix Web Client.

Your workspace may be different depending on your access rights or if your installation has been customized.

Understanding the Workspace

  • Logging on to the Web Client
  • Understanding the Workspace
  • Using List Views

    List views display information in a list with columns and rows of information. Each list view contains a group of records, for example, contacts or accounts. Clicking a column header sorts the information in that column. The arrow marker indicates whether the column is ascending or descending.

  • Using Detail Views

    Most records in Sage SalesLogix display information in both a List view and a Detail view. The Detail view consists of information boxes, tabs and a user-defined middle section.
