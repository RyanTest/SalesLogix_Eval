General/Search Options

Use the General/Search tab to specify such settings as which view opens by default when you first log into Sage SalesLogix Web Client, and who to use as the default owner for any new records you create. You can also set options for using Mail Merge, although these will not be visible unless you selected the “Use ActiveMail” check box when you logged in.

To set general and search options

  1. On the status bar, click Options.
  2. Click the General/Search tab.
  3. In the Show on Startup box, select the default view you want to display when you log in to the Web Client, (for example, What's New or Calendar).
  4. In the Default Owner/Team box, click the Find button, select the user or team you want to be the default owner for new accounts, and then click OK.
  5. In the Intellisync Contact Group box, select an ad hoc contact group for the sync process to occur.

    6. If the ad hoc group you want to sync your contacts to does not appear in the list, you must create the SyncSalesLogix group.

  6. If you want to be automatically logged out of the Web Client after a period of inactivity, select the Enable Automatic Logoff option, enter a number in the box, and select Minute(s) from the drop-down list to determine the length of time.
  7. If you want to be notified when you have unsaved changes on a page, select the Prompt before leaving page with unsaved data check box. This option is selected by default. Clear the check box if you do not want to be notified.
  8. Select the Mail Merge Base Templates you want to use when creating a new e-mail, fax, or letter template.
  1. Select the template you want to use when creating a new template from the from the Select a Template dialog box.
  2. Click OK.
  3. Click the E-mail Base Template, Letter Base Template, or Fax Base Template Find button.
  1. In the Write Menu - Recent Templates box, enter the number of templates you want to display in the most recently used templates list.
  2. In the Fax Provider box, click the drop-down arrow and select the fax provider you want to use for faxing.
  3. For Multi-Currency only: To select a currency type, click the My Currency Findbutton, select a currency from the list, and then click OK.
  4. In the Log to History box, select Prompt if you want the Complete E-mail dialog box to display before an e-mail message is recorded to history. If you want the message to be recorded to history without prompting you, select Do Not Prompt.
  5. Select the Prompt for Duplicate Contacts check box, if you want to be prompted when a duplicate contact is detected.
  6. Select the Prompt if Contact not Found check box, if you want to be prompted when a contact is not found in the database.
  7. If you are using Mail Merge features, select the Use ActiveMail check box.

  8. For Disconnected Web users only: To run reports while disconnected, ensure the Use ActiveReporting check box is selected.
  9. Click Save.
