Logging in to the Sage SalesLogix Web Client

Before you begin, locate your User ID and password. If you do not have this information, contact your system administrator.

If your Web Client connection has expired or you have logged out of the Web Client, the logged off screen appears. To return to the log in screen, click Log in to the SalesLogix Web Client.

To log on

  1. Connect to your company's Web Client. If you have Windows Authentication configured, then you will not have to complete the remaining steps, but will be automatically logged on the Sage SalesLogix Web Client.

This can be done several ways, depending on your browser. In many browsers, it is faster to set the initial screen as the default screen of your browser, or to add the initial screen to a list of favorite sites.

  1. Type your User Name and Password.

Passwords are case-sensitive. In addition, your system administrator may set restrictions on using blank passwords, your user name as a password, and other options. If necessary, see your system administrator to discuss your password options.

  1. Select one of the following Active Mail options:

    		2.

    This applies to the Sage SalesLogix Web Client only.

  2. Select Use my settings to use the ActiveMail setting you selected in your user options.
  3. Select the Use ActiveMail this time (requires ActiveX) to download the Active Mail features, regardless of the setting in user options.
  4. Select the Do not use ActiveMail this time if you do not want to download the Active Mail features, regardless of the setting in user options.
  5. Click Log On.
