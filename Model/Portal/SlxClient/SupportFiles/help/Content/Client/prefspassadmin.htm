Changing a Password (Administrator)

You have the ability to change a user's password through the User Options in the Web Client. Once you have made the changes you want, you must log off, and then log back in for changes to take effect.

To change your password

  1. Click Options on the menu bar.
  2. Click the Password tab, and select the user you want in the list provided.
  3. Enter the new password.
  4. Reenter the new password in the Confirm Password field.
  5. Click Save. A message appears stating the password successfully changed.
