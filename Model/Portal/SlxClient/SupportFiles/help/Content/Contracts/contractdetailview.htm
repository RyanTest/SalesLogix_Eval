Using the Contract Detail View

Use the Detail view to see, add or edit information for a selected record. The Detail view consists of information boxes, tabs and a user-defined middle section.

You can copy information from an existing contract to create a new one. If you use the Copy button, Sage SalesLogix copies information from the original record.

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

How Do I?

Open the Contract Detail View
  • On the Navigation Bar, click Contracts to open the Contract List view. If necessary, use the Lookup to locate a contract, and then click the Reference Number.

Use the Lookup to search for records

View a group

Switch to the List view
  • On the toolbar click the List View button.

The contract information boxes contain the primary information about the contract. Users with appropriate permissions can edit these boxes.
