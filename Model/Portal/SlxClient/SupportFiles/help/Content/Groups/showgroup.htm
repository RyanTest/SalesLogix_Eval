Showing Hidden Groups

You can show a group that has been hidden.

To show a hidden group

  1. From the appropriate list view, on the Group menu, click Manage Groups.
  2. Select the check box(es) of the group(s) you no longer want to hide, or if you want to show all of the groups, select the Select All check box.
  3. Click OK.
