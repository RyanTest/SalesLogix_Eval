Exporting a Group to Excel

You can export groups to Excel. The group column headings and formatting will also be exported.
  • Export to Excel requires the use of Active X Mail components. Active X is not supported by Firefox; therefore, this feature is only available with Internet Explorer.
  • If the Export to Excel option is unavailable, you will need to download the ActiveMail components. The ActiveMail components are downloaded when you log on to the Web Client. In order to download these components, you can set the ActiveMail option in Options, or on the Sage SalesLogix Web Client login page.

To export a group to a file

  1. Open the group that contains the records you want to export.
  2. On a List view, right-click the group tab and click Export to Excel.

Microsoft Excel opens with the exported group. If the group contains currency information, Excel also creates a Pivot table. Excel also adds a total row to every column that contains currency information, so that you can view the group total.

You may receive an error if you try to export a group that includes a data field with a large amount of data.
