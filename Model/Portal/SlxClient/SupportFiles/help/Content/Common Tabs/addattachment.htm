Adding an Attachment

You can add an attachment to a record. All files are stored in the shared attachments directory and synchronize to and from remote databases.

By default, you cannot attach files that are larger than 4MB. You will receive an error unless your administrator has customized your settings.

To attach a file

  1. Open the Attachments tab.
  2. Click the Add button.
  3. Click Browse.
  4. Navigate to the file you want to add, and then click Open.
  5. In the Description box, enter a description for the attachment if the default description is not clear.
  6. Click Save File. You can only add the attachment if you have defined both a description and the file to attach.

To attach a URL

  1. Open the Attachments tab.
  2. Click the Add URL button.
  3. In the URL box, type the complete URL address.
  4. In the Description box, enter a description for the attachment if the default description is not clear.
  5. Click Save File. You can only add the attachment if you have defined both a description and the file to attach.
