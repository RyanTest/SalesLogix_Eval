Selecting an Alternate Address

You can use the Select an Address dialog box to select an existing address for a new contact. The Select another Address button is available only if existing addresses are available. For example, if you are adding a contact to an account with multiple addresses, the account's addresses are available.

To select an alternate address

  1. From the Contact/Account Information dialog box, click the Select another Address button, if it is available.
    The Select an Address dialog box opens.
  1. Click the address that you want to use.
  2. Click Save.
