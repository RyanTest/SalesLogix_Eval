Contact Information Boxes

The contact information boxes contain in-depth information about the selected contact. You can add, edit or delete the information as needed.

 

Box

Description/Action

Contact

Contact's full name. Click the Edit button to open the Edit Name dialog box.

Account

Account to which the contact is linked. Type the name of the account, or click the Use Existing Account link to find an account.

E-mail

Contact's e-mail address. Click in the box and type any changes. To send an e-mail message to the contact, click the E-mail button.

Web

Web address for the account. Click in the box, and type any changes. To open the contact's Web site, click the WWW button.

Work Phone

Contact’s work telephone number. Click in the box and type any changes.

Title

Contact's title within the account's company. Click the ellipsis button and select an item from the list.

Address

Address or location of the contact. Click the Edit button, to open the Edit Address dialog box.

To view a map of the address, click the MapQuest button.

Home Phone

Contact’s home telephone number.

Mobile

Contact’s mobile telephone number.

Fax

Contact’s fax telephone number. Click in the field and type any changes.
