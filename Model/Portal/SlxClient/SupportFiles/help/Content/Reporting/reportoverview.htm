What are Reports?

Reports enable you to view and analyze information in the Sage SalesLogix database. New installations of Sage SalesLogix are integrated with Crystal Reports XI. See your administrator for further information.

Sage SalesLogix Web Reporting enables you to:

  • Share report filters.
  • Filter reports to limit the data included.
  • Export and print reports.

To open the Reports view

  • On the Navigation Bar, click Reports.

To run Web reports in a disconnected state, the disconnected Web Client must run on Window 2003 Server with IIS. The SalesLogix Web Reporting Server and Crystal Reports Server must also be installed and configured. If these requirements are not met, disconnected Web Client users can only run Web Reporting if they have a connection to the Host Web Reporting Server. These users will see reports that represent data from the host database and not their remote database.
