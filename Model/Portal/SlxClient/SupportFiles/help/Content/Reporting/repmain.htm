Report Preview

When you run a report, you can choose to preview it in the Sage SalesLogix Crystal Report Viewer before printing it. From the Viewer, you can print or export, adjust your on-screen view, and search for text in a report.

You can only see reports that have been released to you or to a team that you belong to. You cannot save reports in the Sage SalesLogix Web Client.

How Do I?

View or print a detail report from a detail view

Print a report from the viewer
  1. Select the report you want to preview, then click View Report. The report appears in the SalesLogix Crystal Report Viewer. Depending on the report selected and the number of results produced, a tree view may appear on the left of the window. The tree view lists the pages in the report in alphanumeric or chronological order.
  2. To view a specific page of the report, click the plus sign to expand the list, and then click a name in the tree view.
Export a report from the viewer
  1. Click the Export button on the toolbar. The Export the Report dialog box displays.
  2. In the File Format drop-down box, select the format and destination of the report.
  3. Select one of the Page Range options for the number of pages you want to export. Select All to print the entire report, or select the Pages option and enter the first page number in the From box, and then enter the last page number in the To box.
  4. Click OK.
Print a report from the viewer
  1. Click the Print button on the toolbar.
  2. Select one of the Page Range options for the number of pages you want to print. Select All to print the entire report, or select the Pages option and enter the first page number in the From box, and then enter the last page number in the To box.
  3. Click OK. Acrobat Reader opens and displays the report you selected.
  4. Click Print. If the report does not print correctly, you may need to reformat it for your printer. The report orientation settings will override the orientation in the printer settings.
Show/hide the group tree

Depending on the report selected and the number of results produced, a tree view may appear on the left of the window. The tree view lists the pages in the report in alphanumeric or chronological order.

  • In the Report Viewer, click the Show/Hide Group Tree button.
Page through a report
  • In the Report Viewer, click the First, Prev, Next, or Last buttons to move through multiple pages of a report.
Search text in a report
  1. In the Report Viewer, in the Find box, type search criteria.
  2. Click the Find (binoculars) button.
