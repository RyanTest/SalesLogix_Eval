Running a Report

To run a report

  1. On the Navigation Bar, click Reports.
  2. Select a report type from the Select the type of report list.
  3. Select a report from the Select the name of the report list. A description of the report appears in the box below. (Once selected, Administrators can view to whom the report is released.)
  4. From the Show records that match list, select one of the following:
  • All Records - Includes all records for the report type selected. To filter your selection further, click Edit.
  • Current Group - (For Contacts, Opportunities, Tickets, Defects, and RMA report types) Includes data only for the currently selected group record on the Main View. To filter your selection further, click Edit.
  1. Click View Report. If the group filter includes Unicode data, you will be prompted to remove the group from the filter or select a different group.

    2. The report displays in the Sage SalesLogix Crystal Reports Viewer.
