Viewing Condition Formulas

Administrators familiar with Extended Markup Language (XML) or SQL can view the report condition formulas to assist them in troubleshooting. This option can be helpful for troubleshooting.

To view condition formulas

  1. From the Report view, select a report filter, then click Edit.
  2. Under Conditions, do one of the following:
  • Click View XML to display the report's XML code.
  • Click View SQL to display the report's SQL code.
  1. Click OK.
