Writing and Sending a Fax Message

The following describes how to write and send a fax to a single contact or lead. If you need to send a fax to more than one contact or lead, use Mail Merge.

To write and send a fax message

  1. On the Write menu, point to Fax using Template, and then select a template from the list of recently used templates, or click More Templates to select another template. If prompted to add the template to the most recently used templates list, click Yes to add the template to the list, otherwise click No.
  1. If you have not already selected a contact, find a contact when the Contact Lookup dialog box opens, and then click OK.
  2. If prompted, choose an address from the Select Type of Address dialog box.
  1. If necessary, in the Fax Editor dialog box, make any edits to the template.
  2. Close the Fax Editor.
  3. In the Fax Options dialog box, type a subject in the Subject box.
  4. In the Cover Page list, do one of the following:
  5. Keep the default cover page that appears.
  6. Click the arrow, and then click a different cover page.
  7. Click the arrow, and then click No Cover Page.
  1. If you want to include a message on the cover page, do one of the following:
  2. For a short message, type the message in the Message box.
  3. For a long message, click Message. In the Edit Message dialog box, type your message, and then click OK.
  1. (Optional) To change the default delivery schedule for the fax, click Delivery Options. After setting the delivery options, click OK.
  2. Click Send.
  3. Use the Complete an Activity dialog box to edit the history information, type notes, or schedule a follow-up activity, and then click Complete.
     
  4. If you scheduled a follow-up activity, complete the Schedule an Activity dialog box, and then click Save.

For more information about Mail Merge, refer to the mail Merge Online help system.
