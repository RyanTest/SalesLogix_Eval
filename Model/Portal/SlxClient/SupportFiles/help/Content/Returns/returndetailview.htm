Using the Return Detail View

Use the Detail view to see, add or edit information for a selected record. The Detail view consists of information boxes, tabs and a user-defined middle section.

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

How Do I?

Open the Return Detail view
  • On the Navigation Bar, click Returns to open the Return List view. If necessary, use the Lookup to locate a return, and then click the Return ID.

Use the Lookup to search for records

View a group

Switch to the List view
  • On the toolbar click the List View button.

information boxes

The return information boxes contain the primary information about the return. Users with appropriate permissions can edit these boxes.  

detail view tabs

Each tab shows specific information you can use for return related activities. Click the links in the grid below to see more information about each tab.
