Editing Attachment Information

Use this view to alter the attachment description and upload a new version of an attachment. Attachments synchronize.

By default, you cannot attach files that are larger than 4MB. You will receive an error unless your administrator has customized your settings.

To edit attachment information

  1. Open the Attachments tab.
  2. Select the attachment that you want to edit, and click Edit.
  3. If necessary, update the Description.
  4. Do one of the following:
  • To upload a different file, click Browse, and browse to the new file you would like to upload.
  • To link to a different URL, in the Save File box type the new URL.
  1. Click Save.
