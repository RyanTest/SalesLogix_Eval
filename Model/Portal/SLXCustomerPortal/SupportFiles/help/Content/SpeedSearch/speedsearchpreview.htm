Previewing SpeedSearch Results

After you perform a search and select an item, use the Preview to view detailed information from the item.

To quickly find relevant information

  1. Perform a SpeedSearch and click the Search button.
  2. If necessary, use the First Page, Previous Page, Next Page and Last Page buttons to view the returned records and find the record you want to preview.
  3. Click Preview (red link below record)
  4. Do one of the following:
  • Click Locate first hit to go to the first instance of a keyword.
  • Use the Previous Keyword and Next Keyword buttons on either side of a keyword to go to the previous or next instance of the keyword.
  • Use the inner scroll bar to scan the record or file.

To mark a result as used

If you find an item helpful, but are not inserting the results, do the following:

  • Select the item and click Mark this item as useful. This helps the system keep track of which items are frequently used.
