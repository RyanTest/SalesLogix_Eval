Comments Tab

Use the Comments tab on the Ticket Detail view to add comments for ticket. Once added, your comments appear as an activity for the ticket. A Customer Service representative, using the Sage SalesLogix Client, can edit the ticket's activity while working on resolving the problem.

To add comments to a ticket

  1. On the Navigation Bar, click Tickets.
  2. Use the Tickets lookup to find the ticket.
  3. Select the ticket.
  4. On the Ticket Detail view, click the Comments tab.
  5. In the Comments text box, enter your comments for the ticket.
  6. Click Submit.

    7. The comments appear in the area below the tabs.
